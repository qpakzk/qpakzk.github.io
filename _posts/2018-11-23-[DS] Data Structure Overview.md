# Data Structure Overview

Computer Science(CS)에서 자료구조(Data Structure)는 데이터를 저장하고 관리하는 틀이다. CS 학부 과정에서 다루는 자료구조의 목차는 다음과 같다.

* Abstract Data Type (ADT)
* Linked List
* Stack
* Queue
* Sorting
* Tree
* Graph
* Hash

각각의 주제 별로 포스팅을 진행할 예정이다.
